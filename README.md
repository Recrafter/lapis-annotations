# Lapis Annotations

Compile-time annotations and related declarations for Lapis KSP.

## Usage

A Kotlin compile-time library defining annotations and related declarations used by [Lapis KSP](https://github.com/Recrafter/lapis). For detailed installation and usage instructions, refer to the Lapis KSP documentation.
